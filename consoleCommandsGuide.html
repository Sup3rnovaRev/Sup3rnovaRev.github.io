Fast Restart:

Quickly restarts the match.

						
fast_restart
Note: This just restarts the map it doesn't reload it

Map Restart:

Fully restart the map.

						
map_restart
Note: This restarts the map and reloads it

Field of View (FOV)

Manually tweak your desired FOV value.

						
cg_fov 90

Sensitivity

The command is useful for fine tuning of your desired sensitivity.

						
sensitivity 2

Show FPS / Coords

Following command can be used to both display your current FPS but also your coordinates

						
fast_restart

Kicking Players

There are two different commands for kicking players, one simply uses their name to identify who to kick and the other uses their client id (Number). The kick command uses the players name as a parameter (If their name is in different colors, that does not affect it, just type what you see). The other command is clientkick, if you wish to find out players client ID then use the status command.

						
kick xXOptic_PredatorXx
						
						
status
						
						
clientkick 2
