Private Match Guide

A private match is a great way to play private games with friends without the hastle of setting up a dedicated server.

Return to Guide list

Introduction

There are two methods to do this, one using the in-game friends list and the other using the console to connect directly.

Method 1 - (Friends List):

Note: Requires you to have each other added as friends on Steam and to have steam running.

  1. Start iw4x and select 'CREATE GAME' from the main menu and click 'START GAME'
  2. Tell your friends to press the F key on the main menu to bring up the in-game friends list
  3. Tell them to double click your name to join your game (If they hear a chicken noise, see "Failed to join")

Method 2 - (Connect Command):

  1. Go to WhatIsMyIpAddress.com and PM anyone who wants to join your IP Address
  2. Start iw4x and select 'CREATE GAME' from the main menu
  3. Click 'START GAME'
  4. Tell anyone who wants to join, open the console by using the tilde key while at the main menu
  5. Get them to enter the following command, replacing 'HostsIPAddress' with the IP you previously sent him:
    /connect HostsIPAddress:28960

Failed to join

If you followed the above correctly and your friends are unable to join them you the host of the private match will need to port forward in order for them to connect to you.

  • Port Forward port 28960 UDP & TCP
  • You can find how to do this by Googling 'YourRouterName Port Forwarding'

Note: If port forwarding is too complex, just stick to playing public servers, or maybe join an empty server so you have the same experience as a private game.
