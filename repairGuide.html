Repair Guide

A in-depth guide on to repair broken IW4x install.

Return to Guide list

Requirements

  • qBittorrent or another torrent client of your choice. (This tutorial will use qBittorrent)

Getting Started

  1. Locate your MW2 Install, and make note of the folder name and folder path (Your folder name may differ from the example below).


  1. Download the following torrent and open it with qBittorent or your torrent client of choice All-In-One Torrent.
  2. Once prompted with the following screen right click the folder 'iw4x_full_game' and select 'rename'


  1. Rename the folder to the name of your MW2 folder that we made note of in Step 1 (Our Example: Call of Duty Modern Warfare 2)
  2. Now click the browse icon, a explorer window will open prompting us to select a folder


  1. We need to browse to the parent folder of our MW2 install (1 folder back a level). In the example this is D:\Games


As you can now see we have mimicked our existing MW2 folder structure, meaning that the torrent will download into the exact same location we already have MW2 installed at (Our Example D:\Games\Call of Duty Modern Warfare 2). The torrent client will check this folder and discover any existing files, identify files which are broken or missing and redownload them. In effect reparing your game.

  1. Select 'OK' to start the torrent

  1. Once the torrent status changes to Seeding then the repair process is complete

Congratulations, your game is now repaired.
